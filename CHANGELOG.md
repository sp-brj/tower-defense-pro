# Changelog

Все значимые изменения проекта документируются в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.0.0/).

## [3.0.1] - 2025-01-04

### Исправлено
- **Баг со способностями** — способности больше не активируются при клике на кнопки UI
- **Отсутствующий radius** — добавлен radius для способности "Подкрепление"
- **Проверки null** — добавлены проверки для предотвращения крашей
- **Границы canvas** — способности теперь работают только при клике внутри игрового поля

---

## [3.0.0] - 2025-01-04

### Добавлено
- **Модульная архитектура** — код разбит на отдельные файлы
- **Герой** — управляемый юнит с автоатакой, перемещение ПКМ
- **Способность "Огненный дождь"** — метеориты с уроном по площади (Q)
- **Способность "Подкрепление"** — временные солдаты (W)
- **Башня "Казарма"** — спавнит солдат для блокировки врагов
- **Система звёзд** — рейтинг прохождения по оставшимся жизням
- Улучшенный туториал с информацией о герое и способностях

### Изменено
- Рефакторинг: разделение на модули (config, entities, systems, ui)
- Увеличен размер игрового контейнера
- Обновлённый UI нижней панели с кнопками способностей

### Исправлено
- Корректная работа сохранения настроек
- Правильный расчёт кулдаунов способностей

---

## [2.0.0] - 2025-01-03

### Добавлено
- **Главное меню** с кнопками Играть, Настройки, Рекорды
- **Выбор сложности** — Лёгкий, Нормальный, Сложный, Кошмар
- **Система звуков** — Web Audio API, процедурная генерация
- **Меню паузы** — продолжить, рестарт, выход
- **Туториал** — пошаговое обучение для новичков
- **Настройки** — громкость, автоволна, показ радиуса
- **Таблица рекордов** — топ-10 результатов в localStorage
- **Система очков** — множители за сложность
- **Управление скоростью** — x1, x2, x3
- **Горячие клавиши** — 1-5 башни, Space волна, Esc пауза
- **Башня "Лазер"** — непрерывный луч урона
- **Враг "Целитель"** — лечит ближайших врагов
- **Экран результатов** — статистика после игры

### Изменено
- Полностью переработан UI
- Улучшена визуализация башен и врагов
- Оптимизирована производительность

## [1.0.0] - 2025-01-02

- Система волн с прогрессией сложности
